A simple example combining:

   - Node.js
   - Express
   - Jade
   - jQuery
   - D3


To run it:

   - Run:
      - node ./app.js


It shows

   - How to build an interactive graph with the force layout


Important files:

   - ./app.js -> The file with the source code for the express 's application
   - ./public/javascripts/app.js -> The file with the source code for the client-side application (here's all the D3/jQuery stuff)
